# Simts - Ogres 1. vidusskola
Laravel application that is designed for ZPD (Scientific Research Work) that collects and displays school's project - _100 labie darbi_.

### Requirements
- Composer
- NPM

### Setup
1. Fork the repo and clone it
2. `composer install`
3. `npm install`
4. `npm run dev`
5. `php artisan serve`
6. Get working!
